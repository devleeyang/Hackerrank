
리양의 hackrrank 도전기
===

> 30일동안 알고리즘 문제를 매일 푸는 도전기

## 환경

Language : Swift.  
IDE : XCODE playground  

## 구조
```
Hackrrank.playground 
  │
Pages
  │
Day.xcplaygroundpage ──┰── Contents.swift
		       └── Resources ── Problem.md
```
- Day : 매일 출제되는 문제의 페이지  
- Contents.swift : 문제 해결 방법  
- Problem.md : 문제
